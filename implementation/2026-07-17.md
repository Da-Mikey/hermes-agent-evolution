## Issue #36 — [IMPROVEMENT] Prompt Compression for Token Cost Reduction  
**Status: 🚀 Implemented — PR created**

**Implemented component:** Instruction distillation (system prompt → shorthand compression)

The recursive summarization component already exists in `agent/context_compressor.py` (3452 lines). What was missing was **instruction distillation** — a deterministic, mapping-based system prompt compressor that reduces verbose guidance sections into concise shorthand.

**New file:** `agent/instruction_distiller.py`
- Deterministic mapping-based compression (no LLM calls — zero latency)
- Compresses identity preamble, tool guidance, enforcement text, platform hints
- Gated behind `evolution.prompt_compression.enabled` config flag (disabled by default)
- Config wire: reads from `config.yaml` section `evolution.prompt_compression`
- Target: ~50-60% token reduction on the stable prompt tier

**Tests:** `tests/agent/test_instruction_distiller.py`

**PR:** Created at `evolution/issue-36-instruction-distillation` → base `main`
**Label:** `accepted` added to issue #36
